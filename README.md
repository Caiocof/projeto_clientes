## Iniando o projeto

- Execute "pip install -r requirements.txt"
- Depois "python manage.py runserver"
- Abra http://127.0.0.1:8000/admin (user:admin, password:admin)


## Objetivo

- Explorar algumas formas de validação de campos para um CRUD utilizando o django
